# Scroll of System Amendment

A scroll to perform the rite of system amendment—clean execution when you know what to change.

**Purpose:** Direct modification of the magic system without forced ceremony. When you know what needs changing, this charm provides supported execution.

**Invocation:** `@meta/amend`

---

## Operational Guidance

### For the Spirit

When this scroll is invoked:

1. **Understand the amendment** - What needs to change? Why?
2. **Assess scope** - What components are affected?
3. **Propose execution plan** - How to implement cleanly
4. **Execute with approval** - Make changes after Mage's sanction
5. **Verify completeness** - Confirm amendment fully implemented

No forced ceremony—just clean, supported system modification.
