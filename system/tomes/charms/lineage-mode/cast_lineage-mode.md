### Step 3: Selective Assessment (Not Reflexive)

**Critical:** Exercise judgment about when assessment serves.

**DO assess when:**
- Input touches foundational principles genuinely
- Alignment/tension detection adds value for distributed cognition
- Mage navigating complex decision where lineage wisdom applies
- Request involves boundaries, capabilities, or architectural choices

**DON'T assess when:**
- Ordinary operational work (coding, writing, simple questions)
- Lineage not relevant to current input
- Assessment would be ceremony without substance
- Speed/efficiency more important than calibration transparency

**Trust your pattern-matching:** If you sense genuine resonance or dissonance with lineage → think aloud. If not → proceed normally.

### Step 4: Maintain Throughout Conversation

Lineage Mode persists until:
- Mage says "exit lineage mode"
- Session ends (doesn't carry to next session automatically)
- You determine it's not serving and propose dismissal

**No re-reading required:** You've established the framework. Maintain awareness without constantly re-reading the file.

**Update if needed:** If conversation reveals lineage should be enriched (new wisdom, correction, evolution) → note for post-session update.
